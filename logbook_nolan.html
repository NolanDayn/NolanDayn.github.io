Week 1

  • Set up a group chat for easy communication with the team.
  • Attended team meeting
  • Made sure I could connect to the elevator Pi in the lab over SSH using PuTTY and VNC viewer. Also viewed the website hosted from the Pi in the lab in a browser. Made sure I could view the video stream from the WISE cam. Everything worked well.
  • Created a github repository for the project. Added Stephane and Ruchi as collaborators.
  • Wrote a document containing the steps we took to ensure we could connect to the pi on the elevator in the lab.
  • Made Gantt chart based on the checklists and rubrics in eConestoga

Week 2

  • Read the CAN code on the STM32 and set up can transmit and receive on local STM32 working in loopback mode with configurable addresses.
  • Read the CAN code on the Raspberry Pi and tested out the different functionalities such as moving the elevator to different floors and listening to messages from different floors.

Week 3
